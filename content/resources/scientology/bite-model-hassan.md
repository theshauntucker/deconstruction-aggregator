# BITE Model / Freedom of Mind

Psychologist Steve Hassan developed the BITE Model as a framework for analyzing undue influence, coercive control, and thought reform. The model categorizes control into four dimensions: Behavior (regulating what members do), Information (controlling what information is available), Thought (shaping how members think), and Emotion (manipulating emotional states). This framework is foundational to understanding Scientology's mechanisms.

Hassan's Freedom of Mind website provides free educational resources, including PDF guides explaining the BITE Model's application to various high-control groups. The model has become standard in cult research and exit counseling. Using this analytical framework alongside specific Scientology documentation provides rigorous language for describing the organization's coercive practices.
